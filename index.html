Initiation au language HTML et CSS

  • Introduction

    • HTML (Hypertext Markup Language) est un language crée en 1991 et fonctionnant sur un système de balisage afin de coder la structure et le contenu d'une page web. CSS (Cascading Style Sheets) est un language complémentaire fonctionnant sur un système de "feuilles de style" et permettant une gestion dissociée de l'affichage de la page web.

    L'association de HTML et de CSS permet une grande maniabilité dans le domaine du développement web en permettant la modification du contenu sans affecter l'affichage et vice versa.
