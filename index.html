Initiation au language HTML et CSS

  • Introduction

    • HTML (Hypertext Markup Language) est un language crée en 1991 et fonctionnant sur un système de balisage afin de coder la structure et le contenu d'une page web. CSS (Cascading Style Sheets) est un language complémentaire fonctionnant sur un système de "feuilles de style" et permettant une gestion dissociée de l'affichage de la page web.

    L'association de HTML et de CSS permet une grande maniabilité dans le domaine du développement web en permettant la modification du contenu sans affecter l'affichage et vice versa.

  • Base du Document HTML

  • < html > : Balise principale

    < head > : En tête de la page

    < body > : Corps de la page

  • En tête entre les balises < head > et < /head >

  • < link / > : Liaison avec une feuille de style

    < meta / > : Metadonnées de la page web (charset, mots clefs, ...etc)

    < script > : Code JavaScript

    < title > : Titre de la page

    < style > : Code CSS

    • Le texte dans la norme HTML doit être mis dans une balise spécifique, la balise < p > exempledetexte < /p >

    Pour retourner à la ligne, refaire un paragraphe, on réutilise les balises < p > exempledetexte < /p >

  • Exemple de code minimal HTML

    • Le titre de la page ….......... …..........

    Ici le jeu de caractère (charset) utf-8 a été choisi. Il s'agit d'un jeu de caractères standard couremment utilisé.
